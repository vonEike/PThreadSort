1. Введение

В данной расчётной работе рассматривается разработка программы параллельной сортировки с использованием библиотеки pthread языка C.

Цель работы — изучить принципы многопоточности и реализовать алгоритм сортировки с использованием нескольких потоков.

🟩 2. Руководство пользователя
2.1 Назначение программы

Программа предназначена для сортировки массива чисел с использованием параллельных вычислений.

Сортировка выполняется путём:

разделения массива на части
параллельной сортировки частей
последующего слияния результатов
2.2 Формат входных данных

Входные данные представляют собой:

массив целых чисел
может задаваться:
внутри программы
либо через файл (если реализовано)

Пример файла:

5 3 8 1 2

Требования:

числа разделены пробелами
перевод строки в конце файла не обязателен
2.3 Запуск программы
Компиляция
gcc main.c -o sort -pthread
Запуск
./sort

(или с параметрами, если есть)

Пример работы
Исходный массив:
5 3 8 1 2

Отсортированный массив:
1 2 3 5 8

👉 (в отчёт вставить скриншот запуска программы)

2.4 Выходные данные

Программа выводит:

отсортированный массив в консоль
или
файл с результатом (если реализовано)
🟩 3. Руководство программиста
3.1 Используемые структуры

Пример:

typedef struct {
    int* array;
    int start;
    int end;
} ThreadData;

Описание:

array — указатель на массив
start — начальный индекс
end — конечный индекс
3.2 Описание функций
🔹 sort_function
void* sort_function(void* arg);

Описание:
Функция выполняет сортировку части массива в отдельном потоке.

Аргументы:

arg — указатель на структуру с данными
🔹 merge
void merge(int* arr, int left, int mid, int right);

Описание:
Объединяет два отсортированных подмассива.

🔹 main
int main();

Описание:
Главная функция программы:

создаёт потоки
распределяет данные
запускает сортировку
объединяет результаты
3.3 Алгоритм работы программы
Генерация или чтение массива
Деление массива на части
Создание потоков
Сортировка частей массива
Ожидание завершения потоков
Слияние отсортированных частей
Вывод результата
3.4 Особенности реализации
используется библиотека pthread
синхронизация через pthread_join
каждый поток работает со своей частью данных
🟩 4. Заключение

В ходе выполнения работы:

изучена библиотека pthread
реализована параллельная сортировка
исследована эффективность многопоточности
