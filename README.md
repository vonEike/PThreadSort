# 📘 PThreadSort

PThreadSort — это демонстрационный проект на языке C, показывающий, как реализовать **параллельную сортировку** с использованием **POSIX Threads (pthreads)**.

Проект предназначен для обучения и экспериментов с многопоточностью и производительностью.

---

## 📌 Описание

Алгоритм работает по принципу:

1. Делит массив на несколько частей  
2. Сортирует каждую часть в отдельном потоке  
3. Объединяет отсортированные части в один массив  

---

## ⚙️ Как это работает

```

[Исходный массив]
↓
[Разделение на N частей]
↓
[Параллельная сортировка]
↓
[Слияние результатов]
↓
[Отсортированный массив]

````

---

## 🧩 Основные компоненты

### 🔹 Потоки (pthreads)

Используется библиотека:

```c
#include <pthread.h>
````

Создание потока:

```c
pthread_create(&thread, NULL, sort_function, args);
```

Ожидание завершения:

```c
pthread_join(thread, NULL);
```

---

### 🔹 Разделение массива

Массив делится на части:

```c
chunk_size = size / num_threads;
```

Каждый поток получает свою часть массива.

---

### 🔹 Функция сортировки

Каждый поток выполняет:

```c
void* sort_function(void* arg)
```

Внутри:

* получает указатель на данные
* сортирует их (например, через `qsort`)

---

### 🔹 Слияние (merge)

После завершения потоков:

* все части объединяются
* используется логика merge sort

Пример:

```
[1 5 9] + [2 6 8] → [1 2 5 6 8 9]
```

---

## 🚀 Запуск

### Сборка

```bash
gcc main.c -o sort -pthread
```

### Запуск

```bash
./sort
```

(возможны аргументы: размер массива, число потоков)

---

## 🧠 Ключевые идеи

* **Параллелизм** — несколько потоков работают одновременно
* **Divide & Conquer** — разделение задачи на части
* **Синхронизация** — через `pthread_join`

---

## 📊 Производительность

### ✅ Плюсы

* ускорение на многоядерных CPU
* хорошая демонстрация многопоточности

### ❌ Минусы

* накладные расходы на создание потоков
* merge может быть узким местом
* неэффективен для маленьких массивов

---

## ⚠️ Ограничения

* нет балансировки нагрузки
* последовательное слияние
* не оптимизирован для production

---
